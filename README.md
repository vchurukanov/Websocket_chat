# websocket_chat

Простий WebSocket чат на Node.js (сервер + клієнт на чистому JS).

Запуск локально

1. Відкрий термінал у папці проекту:

   cd "C:\\Users\\user\\Downloads\\Telegram Desktop\\ws_chat\\websocket_chat"

2. Встанови залежності:

   npm install

3. Запусти сервер:

   npm start

4. Відкрий у браузері:

   http://localhost:3000

Пуш на GitHub

1) HTTPS (найпростіший):

   git remote add origin https://github.com/YOUR_USERNAME/websocket_chat.git
   git branch -M main
   git push -u origin main

2) SSH (якщо налаштовано):

   git remote add origin git@github.com:YOUR_USERNAME/websocket_chat.git
   git branch -M main
   git push -u origin main

3) GitHub CLI (швидко створює репо і пушить):

   gh repo create websocket_chat --public --source=. --remote=origin --push

Заміни YOUR_USERNAME на свій логін GitHub або використовуй команду з GitHub CLI.
